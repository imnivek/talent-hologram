# 天賦數全息圖

用西元生日畫出「全」字形的天賦結構，看懂自己的主性格與彼此的合作化學。

線上版（GitHub Pages 啟用後）：`https://imnivek.github.io/talent-hologram/`

## 功能
- 個人全息圖：生日層層相加，呈現主性格 O、動機、能量、潛意識與三我（本我／自我／社會我）三層同心圓。
- 兩人配對：以主性格為主軸，看互補、摩擦、分工與合作化學。
- 團隊合作：主性格光譜、角色分佈、共同盲點與互補分組建議。
